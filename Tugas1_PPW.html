Pengantar Web Mining#

Hasil Cek Turnitin#

Definisi Web Mining:#

Web Mining adalah penerapan teknik data mining pada data web untuk mengekstrak informasi, menemukan pola, dan menghasilkan pengetahuan baru. Data yang dimaksud bisa berupa isi halaman web, hubungan antar halaman (link/hyperlink), maupun aktivitas pengguna di web. Tujuannya adalah menemukan pola berguna atau pengetahuan dari struktur web, isi halaman web, dan perilaku pengguna.

Tantangan Pemrosesan data web#

  • Ukuran web yang sangat besar (big data).

  • Data tersedia dalam berbagai format (HTML, XML, teks bebas, gambar, video).

  • Tidak ada domain khsusus, karena web mencakup banyak topik.

  • Dinamika web yang berubah sangat cepat.

Secara umum, Web mining dibagi menjadi 3 bagian utama:#

1. Penambangan isi web (content mining)#

Web content mining adalah proses mengambil informasi yang bermanfaat dari sebuah halaman web. Konten ini bisa berupa :

  • Teks (text mining): Berupa konten tulisan seperti artikel,ulasan,caption atau komentar pengguna.

  • Gambar (image mining): Visual seperti foto, grafik dan ilustrasi digital.

  • Audio (audio mining): Berupa suara rekaman,music atau efek suara.

  • Video (video mining): Konten bergerak seperti video streaming,vlog,documenter.

  • Data terstruktur: Informasi yang tersusun rapi seperti table, daftar, atau database.

Aplikasi yang digunakan dari Text mining meliputi :#

  1. Ekstraksi informasi: Proses mengambil bagian bagian penting dari teks.

  2. Topik modelling: Mencari tahu topik utama pada sekumpulan dokumen.

  3. Peringkasan dokumen: Membuat versi pendek dari teks yang panjang.

  4. Klasifikasi dokumen: Mengelompokkan dokumen ke dalam kategori yang telah ditetapkan sebelumnya (misalnya: identifikasi spam, klasifikasi artikel berita).

  5. Pengelompokan dokumen: Menyusun dokumen ke dalam kelompok-kelompok berdasarkan kesamaan isi tanpa menggunakan label kategori yang telah ditentukan sebelumnya.

  6. Ekstraksi kata kunci: Mencari kata atau frasa yang paling penting dari teks.

  7. Analisa Sentimen: Menilai apakah isi teks bersifat positif, netral, atau negatif. Cocok untuk menganalisis ulasan atau opini publik.

2. Web Usage Mining#

Web Usage Mining adalah proses mengambil informasi yang dilakukan dari data kunjungan halaman dan transaksi pengguna, seperti log akses server, log referer, log agen, cookie, metadata halaman dan konten, serta data clickstream yang mencatat urutan klik pengguna.

Proses ini bertujuan untuk mengenali pola dan kebiasaan pengguna saat mereka menjelajahi situs. Aplikasi yang penting dari Web Usage Mining meliputi:

  • Product Recommendation adalah fitur yang menyarankan produk yang kemungkinan besar disukai pengguna, berdasarkan kebiasaan dan minat mereka atau kesamaan dengan pengguna lain.

  • Personalized Search adalah pencarian yang menampilkan hasil sesuai dengan minat, kebiasaan, dan situasi pengguna.

3. Penambangan Struktur Web (Web Structure Mining)#

Penambangan Struktur Web (Web Structure Mining) adalah proses menganalisis hubungan antar halaman dalam sebuah situs web, biasanya melalui struktur tautan (link). Tujuannya adalah memahami bagaimana halaman-halaman saling terhubung. Aplikasi yang penting dari Web Structure Mining yaitu:

  • Grafik Web: representasi struktur halaman web dalam bentuk jaringan, di mana halaman digambarkan sebagai simpul (node) dan hyperlink sebagai sisi (edge).

  • Identifikasi Simpul Penting: menemukan halaman atau bagian dalam jaringan web yang paling berpengaruh atau sering jadi pusat perhatian.

  • Deteksi Komunitas: mencari kelompok orang atau halaman web yang sering saling terhubung atau berinteraksi satu sama lain, lebih sering daripada dengan orang atau halaman di luar kelompok itu.

Proses Web Mining#

Proses Web Mining ini mirip dengan Data Mining, namun yang membedakan adalah tahap awal pengumpulan dan pengolahan data sebelum dilakukan analisis. Tahapan tahapan tersebut adalah:

  1. Gathering and Exploration (Pengumpulan dan eksplorasi data) adalah tahap awal di mana kita mengambil data dari internet, biasanya dengan alat otomatis seperti web crawler atau lewat layanan Web API. Setelah datanya didapat, kita pelajari isi dengan melihat angka-angka ringkas dan membuat grafik.

  2. Preprocessing and Transformation adalah tahap paling lama, bahkan bisa menghabiskan 7080% waktu dari seluruh proyek. Tujuannya adalah mengubah data mentah menjadi bentuk yang rapi dan siap diproses lebih lanjut. Langkah penting dalam tahap ini:

  • Reduksi dimensi dan Seleksi Fitur

  • Transformasi atribut: Mengubah teks atau atribut lain menjadi angka agar bisa diproses oleh komputer.

  • Mengintegrasikan berbagai sumber data.

  1. Actual Data Mining adalah tahap di mana kita mulai mencari pola atau membangun model dari data yang sudah dipersiapkan. Proses ini dilakukan secara iteratif dengan mencoba berbagai kombinasi hyperparameter, mengevaluasi metode yang berbeda, serta meningkatkan jumlah dan kualitas data pelatihan untuk memperoleh model yang paling akurat.
